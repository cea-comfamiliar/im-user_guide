Gestion de Incentivos

Objetivo​

Desarrollar un aplicativo que permita gestionar la asignación de puntos para distribuir los incentivos entre todos los empleados utilizando el plan de incentivos del periodo institucional.

Para el desarrollo de este aplicativo se utilizaron dos tecnologías para la gestión del frontend y el backend, para la primera se utilizó flutterflow y para la segunda supabase, a continuación se comparten los links de ambas aplicaciones y su documentación.
